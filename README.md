# Pyramid
C++ Program which makes a pyramid out of any string that you give it.

Output examples:

An input of "12345":

       1
      121
     12321
    1234321
   123454321

An input of "1234567890":

            1
           121
          12321
         1234321
        123454321
       12345654321
      1234567654321
     123456787654321
    12345678987654321
   1234567890987654321

An input of "Hello World":

             H
            HeH
           HeleH
          HellleH
         HellolleH
        Hello olleH
       Hello W olleH
      Hello WoW olleH
     Hello WoroW olleH
    Hello WorlroW olleH
   Hello WorldlroW olleH
